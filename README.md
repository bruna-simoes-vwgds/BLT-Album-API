# BLT-Album-API

API for managing songs and albums

## Endpoints

### Album endpoints
 - GET `/albums`
 - GET `/album/id/:id` (receives the 'song' parameter)
 - POST `/album/id/:id` (ID should be autogenerated)
 - DELETE `/album/id/:id`
 - PATCH `/album/`

### Song endpoints
 - GET `/songs` (receives the 'album' parameter)
 - GET `/song/id/:id`
 - DELETE `/song/id/:id`
 - PATCH `/song`
 - POST `/song` (ID should be autogenerated)
 - GET `/songs/album/:album`

1-* relationship -> An album can have several songs, but one song can only be associated with one album

## Structures

### Song structure
 - songID:          UUID
 - songName:        string
 - artistName:      string
 - dateLaunched:    date
 - dateAltered:     date
 - songLength:      timestamp
 - Album:           Album

### Album structure
 - albumID:         UUID
 - albumName:       string
 - artistName:      string
 - dateLaunched:    date
 - dateAltered:     date
 - songs:           []Song
